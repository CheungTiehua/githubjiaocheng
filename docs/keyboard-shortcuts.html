GitHub 快捷键大全

在 GitHub 任意页面按 ? 可查看官方快捷键列表。 以下为最常用快捷键，按场景分类。

仓库页面

快捷键 作用
. 打开 VS Code 网页版在线编辑器
t 快速跳转文件（激活文件搜索）
s 或 / 聚焦搜索框
w 切换分支/标签选择器
l 跳转到指定行（在文件查看页）
b 打开 Blame 视图（在文件查看页）
y 将 URL 转换为永久链接（固定到当前 commit）

页面跳转（g + 字母）

快捷键 跳转到
g c Code 标签
g i Issues 标签
g p Pull requests 标签
g a Actions 标签
g b 仓库主页
g w Wiki 标签
g s Settings 标签
g d Discussions 标签

Issue / PR 页面

快捷键 作用 位置
c 创建新 Issue / 新 PR Issue/PR 列表页
r 引用回复（光标在评论框时） Issue/PR 详情页
e 编辑当前 Issue/PR 标题或正文 Issue/PR 详情页
Ctrl + Enter 提交评论 评论框
Ctrl + Shift + Enter 提交评论并关闭 Issue Issue 评论框
⌘ + B（Mac）/ Ctrl + B Markdown 加粗 编辑器
⌘ + I（Mac）/ Ctrl + I Markdown 斜体 编辑器

通知页面

快捷键 作用
e 标记当前通知为已读
Shift + e 标记所有通知为已读
j 下一条通知
k 上一条通知
o 或 Enter 打开当前通知

代码浏览

快捷键 作用 位置
t 文件查找 仓库页
l 跳转到行 文件查看页
w 切换分支 仓库页
b Blame 视图 文件查看页
y 永久链接 文件查看页
. VS Code 编辑器 仓库任意页
i 切换编辑和查看模式 文件查看页（可选）

全站通用

快捷键 作用
? 显示所有快捷键
Esc 关闭当前弹窗 / 取消聚焦 / 取消选中
Ctrl + K（Mac: ⌘ + K） 打开命令面板

快捷键速查卡（打印版）

仓库页:
  t = 搜索文件    w = 切换分支
  . = VS Code打开 g c = Code
  g i = Issues    g p = PR
  g a = Actions   s = 搜索框

文件页:
  l = 跳行        b = Blame
  y = 永久链接

通知:
  j/k = 下/上     o = 打开
  e = 标已读

全站:
  ? = 全部快捷键  Esc = 关闭
